## HTTP Session

### Session 이란?
* 클라이언트(사용자)가 서버에 접속을 하면 그 클라이언트만의 저장공간에 서버에 생성이 되는데 이 공간이 바로 Session입니다.
* Session은 접속한 클라어언트만의 고유공간이기 때문에, 서버에 접속한 클라이언트 숫자만큼 Session이 생성됩니다.
* Session은 클라이언트의 브라우저 단위로 생성됩니다.
